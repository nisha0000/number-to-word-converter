# number-to-word-converter


## Front end

To build this converter, navigate to `frontend` directories. Then,

1. install the packages with `npm install`
2. Run the application with `npm run start`


To run the tests, use `npm run test`
To open storybook, use `npm run storybook`



## Back end

To build this converter, navigate to  `backend` directories. Then,

1. install the packages with `npm install`
2. Run the application with `npm run start`


To run the tests, use `npm run test`
